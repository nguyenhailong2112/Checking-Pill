# Detection YOLO Dataset

This folder stores generated configs/manifests for the one-class pill detector.

The actual image and label files remain in the source archive folder unless a
copy/export step is intentionally performed.

Expected source layout:

```text
archive/
  images/
    train/
    val/
  labels/
    train/
    val/
```

Class mapping:

```text
0: pill
```
